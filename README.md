# CrazyTank 
A game imitate battle city,support multiplayer on LAN,unfinshed work.

高仿90坦克大战，支持局域网多人联机，未完成品。

## 使用说明
先要打开 TankServer.exe 启动服务器。

![](Images/tankServer.png)

再启动游戏 CrazyTank.exe。

![](Images/tankWar.png)
